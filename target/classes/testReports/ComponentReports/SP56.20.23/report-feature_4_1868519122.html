Feature DOM-Content-Service: POST packages - in order to create a flashware
DOM CONTENT SERVICE is the main actor in building the flashware for a car. It builds both online and offline flashware and it also makes use of other services and their data in order to bring up all the data relevant to the car together and build the databases and the content which are going to be used on the car. Afterwards it will notify all the subscribers and it uploads both DOM.zip and OA.zip to the Artifactory
